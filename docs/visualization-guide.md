# 시각화 기능 가이드

## 개요

ICF Core Set Visual 페이지를 참고하여 구현된 시각화 기능은 평가 결과를 직관적으로 이해할 수 있도록 도와줍니다.

## 주요 기능

### 1. 전체 요약 시각화
- **전체 평균 점수**: 모든 평가 항목의 평균 점수를 큰 숫자로 표시
- **색상 코딩**: 점수에 따라 색상이 자동으로 변경
  - 녹색 (0-1점): 양호
  - 노란색 (1-2점): 보통
  - 주황색 (2-3점): 주의
  - 빨간색 (3-4점): 심각
- **진행 바**: 전체 평균을 시각적으로 표현

### 2. 도메인별 차트
각 도메인(b, s, d, e)별로:
- **평균 점수 바**: 도메인 전체의 평균 점수를 긴 바 차트로 표시
- **개별 항목 바**: 각 ICF 코드별 점수를 작은 바 차트로 표시
- **항목 개수**: 평가된 항목 수 표시

### 3. 인터랙티브 요소
- **실시간 업데이트**: 평가 점수가 변경되면 시각화가 자동으로 업데이트
- **색상 범례**: 점수 범위에 따른 색상 의미 설명

## 사용 방법

1. 평가 항목에 점수를 입력하면 자동으로 시각화가 나타납니다
2. 도메인별로 결과를 확인하여 강점과 약점을 파악할 수 있습니다
3. 개별 항목의 점수를 비교하여 세부적인 분석이 가능합니다

## 시각화 해석

### 점수 의미
- **0-1점 (양호)**: 해당 영역에서 문제가 거의 없거나 경미함
- **1-2점 (보통)**: 일부 어려움이 있지만 대체로 양호함
- **2-3점 (주의)**: 상당한 어려움이 있어 주의가 필요함
- **3-4점 (심각)**: 심각한 문제가 있어 즉각적인 개입이 필요함

### 활용 방법
1. **치료 계획 수립**: 낮은 점수 영역을 우선적으로 개선
2. **진행 상황 추적**: 시간에 따른 점수 변화를 모니터링
3. **보호자 설명**: 시각적 자료로 평가 결과를 쉽게 설명

## 기술적 세부사항

- **순수 CSS**: 외부 차트 라이브러리 없이 CSS만으로 구현
- **반응형 디자인**: 모바일과 데스크톱 모두에서 최적화
- **성능 최적화**: useMemo를 사용하여 불필요한 재계산 방지

## 향후 개선 사항

- **라인 차트**: 시간에 따른 점수 변화 추적
- **파이 차트**: 도메인별 비율 시각화
- **히트맵**: 여러 평가 결과를 비교하는 히트맵
- **인쇄 최적화**: PDF에 포함될 때 최적화된 레이아웃
