# 평가 항목 선택 기준 및 ICF 매핑 가이드

## 현재 샘플 평가도구의 항목 선택 기준

현재 샘플 평가도구에는 **8개의 평가 항목**이 포함되어 있으며, 모두 **ICF d (활동 및 참여) 영역**에 속합니다.

### 포함된 항목 목록

#### 1. 이동 관련 (d4: 이동)
- **d450: 걷기** - 기본 보행 능력
- **d455: 이동하기** - 다양한 이동 패턴
- **d460: 다양한 장소로 이동하기** - 지역사회 이동

#### 2. 자기관리 (d5: 자기관리)
- **d510: 씻기** - 신체 청결
- **d520: 옷 입기** - 의복 착용/탈의
- **d550: 먹기** - 음식 섭취

#### 3. 대인관계 (d7: 대인관계)
- **d710: 기본적 대인관계** - 친밀한 관계
- **d720: 복잡한 대인관계** - 사회적 관계

### 항목 선택 기준

현재 샘플 항목들은 다음 기준을 기반으로 선택되었습니다:

1. **일상생활 필수 활동 (ADL/IADL)**
   - 일상생활에서 가장 기본적이고 중요한 활동
   - 독립성 평가에 핵심적인 영역

2. **ICF Participation 영역 우선**
   - 요구사항에 따라 Participation(d) 영역을 우선 시각화
   - 실제 사회 참여와 관련된 활동 중심

3. **임상적 활용도**
   - 재활, 작업치료, 물리치료 등에서 자주 평가되는 영역
   - 보호자와 치료사 모두 이해하기 쉬운 항목

4. **평가 가능성**
   - 관찰 가능하고 측정 가능한 활동
   - 0-4점 척도로 평가 가능한 구체적 행동

## ICF 분류체계 구조

### ICF 코드 체계
```
d[카테고리][항목]
예: d450 = d(활동 및 참여) + 4(이동) + 50(걷기)
```

### 주요 카테고리
- **d1**: 학습과 지식 적용
- **d2**: 일반적인 작업과 요구사항
- **d3**: 의사소통
- **d4**: 이동 (Mobility)
- **d5**: 자기관리 (Self-care)
- **d6**: 가정생활
- **d7**: 대인관계 (Interpersonal interactions and relationships)
- **d8**: 주요 생활 영역
- **d9**: 지역사회, 사회 및 시민생활

## 새로운 항목 추가 기준

### 1. ICF 코드 매핑
각 평가 항목은 반드시 ICF 코드와 매핑되어야 합니다:
- **ICF 코드**: WHO 공식 분류체계 코드 (예: d450)
- **ICF 제목**: 공식 분류명 (예: "걷기")
- **매핑 근거**: 왜 이 ICF 코드를 선택했는지 설명

### 2. 평가 문항 작성 기준
- **임상 언어 보존**: 전문 용어를 사용하되 이해하기 쉽게
- **구체적 행동**: 관찰 가능한 구체적 행동 중심
- **평가 가능성**: 0-4점 척도로 평가 가능해야 함

### 3. 선택 옵션 기준
ICF qualifier 체계를 따름:
- **0점**: 문제 없음 (No problem)
- **1점**: 경미한 문제 (Mild problem)
- **2점**: 중간 정도 문제 (Moderate problem)
- **3점**: 심각한 문제 (Severe problem)
- **4점**: 완전한 문제 (Complete problem)

## 항목 선택 프로세스

### 1단계: 평가 목적 정의
- 어떤 집단을 평가하는가? (예: 뇌졸중 환자, 노인 등)
- 어떤 영역을 중점적으로 평가하는가?
- 평가 결과를 어떻게 활용할 것인가?

### 2단계: ICF 코드 후보 선정
- ICF 분류체계에서 관련 코드 검색
- 평가 목적에 부합하는 코드 선택
- 중복되거나 유사한 코드 통합 검토

### 3단계: 평가 문항 개발
- ICF 코드를 구체적인 평가 문항으로 변환
- 임상 언어로 설명 추가
- 각 선택지에 구체적 설명 추가

### 4단계: 매핑 근거 작성
- 왜 이 ICF 코드를 선택했는지
- 문항과 ICF 코드의 연결 근거
- 평가 목적과의 연관성
