## 항목 추가 방법

새로운 항목을 추가하려면 `data/sample-assessment.ts` 파일을 수정하세요:

1. `domains.d.codes` 배열에 ICF 코드 추가
2. `items` 배열에 평가 항목 추가
3. `icfCode`로 코드와 항목 연결
4. `mappingRationale`에 매핑 근거 작성

### 예시: 새 항목 추가
```typescript
{
  id: "item-d630",
  icfCode: "d630",
  domain: "d",
  question: "식사 준비 능력은 어떻습니까?",
  clinicalDescription: "...",
  mappingRationale: "...",
  options: [...]
}
```
